# Airline Yield Manager

## Problem Statement and Outcome

**Problem:** Airlines need dynamic pricing models to adjust fares based on capacity, days to departure, and competitor prices. 
**Outcome:** This application trains a Random Forest Regressor on synthetic flight data, tunes hyperparameters via GridSearchCV, saves `model.joblib`, and runs a Streamlit front end for dynamic fare recommendations and a live pricing feed. 

The application is configured to run fully locally for inference, and is ready for deployment on **Render**.

## Quick Start

### 1. Set up the Environment

Clone the repository and install dependencies:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Train the Model

Generate synthetic data and train the model locally. This will produce `01_dataset/flight_data.csv` and `03_starter_kit/model.joblib`.

```bash
python 03_starter_kit/train.py
```

### 3. Run the Streamlit UI Locally

Start the local Streamlit application:

```bash
streamlit run 06_build_app/app.py
```

- **Scenario Planner**: Input flight details to receive a dynamic fare recommendation.
- **Live Feed**: Streams `flight_data.csv` to demonstrate bulk inference.

## Deployment (Render)

This project is configured for seamless deployment on Render using the provided `render.yaml`.

1. Push your repository to GitHub/GitLab.
2. In your Render Dashboard, click **New +** and select **Blueprint**.
3. Connect your repository. Render will automatically detect `render.yaml` and deploy the Streamlit Web Service using the provided `Dockerfile`.
